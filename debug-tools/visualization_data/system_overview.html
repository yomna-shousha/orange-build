🍊 Orange Build - System Architecture Overview

Orange Build is a sophisticated AI-powered webapp generation platform that transforms natural language descriptions into fully functional web applications through a phase-wise development process with real-time streaming and parallel processing.

• Multi-AI Model Strategy: Different AI models optimized for specific tasks

• Real-time Communication: WebSocket broadcasting for live updates

Corrected Implementation Details:
• SCOF Processing: SCOF parser processes the streaming LLM output during Phase Implementation
• Realtime Fixing: RealtimeCodeFixer spawned immediately per file, runs in parallel
• Deployment Timing: deployment_started/completed events occur after phase validation and after file regeneration
• File Regeneration: Individual file deployments happen during review cycles after regeneration
• Promise Coordination: All files fixed in parallel, then batch deployment after validation
• Issue Tracking: Runtime errors cleared after deployment, fed to next phase planning
• Review Cycles: Up to 10 review cycles with targeted file regeneration and individual deployment
• State Persistence: generatedFilesMap and generatedPhases updated after phase completion
